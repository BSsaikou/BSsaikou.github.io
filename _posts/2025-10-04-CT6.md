## 모듈

import 모듈 로 모듈을 불러올 수 있다.

또는 from 모듈 import 이름 

```python
import math

a = math.sqrt(2) # 2의 제곱근

print(a) # 1.4142135623730951

pi = math.pi # pi 값은?

print(pi) # 3.141592653589793
```
