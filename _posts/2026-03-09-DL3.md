```python
import numpy as np

A = np.array([1,2,3,4])

print(np.ndim(A)) # 1      1차원 배열

print(A.shape) # (4, )       

B = np.array([[1,2],[4,5],[7,8]])

print(np.ndim(B)) # 2   2차원 배열

print(B.shape) # (3, 2)     3x2
```

## 행렬의 내적


```python
A = np.array([[1,2],[3, 4]])

B = np.array([[5, 6], [7, 8]])

print(np.dot(A, B)) # ([[19, 22], [43, 50]])

```

### 내적을 하려면 대응하는 차원의 원소 수를 일치시켜야 한다


```python
A = np.array([[1,2,3],[4,5,6]])

B = np.array([[1,2],[3,4],[5,6]])

C = np.array([[1,2],[3,4]])

print(np.dot(A,B)) # 2x3 3x2 배열 내적 가능

print(np.dot(A,C)) # 2x3 2x2 배열 내적 불가능

```
