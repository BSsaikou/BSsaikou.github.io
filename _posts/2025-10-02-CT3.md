## 자료형


```python
print(type(1)) # <class 'int'>

print(type(1.2)) # <class 'float'>

# 5 / 3 을 연산하면 1.666666... 의 float 값이 나오게 된다

print(type(3 + 4j)) # <class 'complex'>

# 파이썬에서는 허수 i 를 j 로 나타낸다. 
# 이러한 복소수가 있는 자료형은 complex로 나타난다.

print(type('A')) # <class 'string'>

print(type([1, 2, 3])) # <class 'List'>

print(type(('one', 'two', 'three'))) # <class 'tuple'>

print(type(type(1))) # <class 'type'>

print(type({'one': 1, 'two': 2, 'three': 3})) # <class 'dict'>

# 키 와 값의 짝으로 이뤄진 자료형을 딕셔너리라고 하고 이를 매핑이라고 한다.

print(type(1 == 2)) # <class 'bool'>

print(type({'apple', 'banana'})) # <class 'set'>

# 집합을 표현하는 자료형
```
