## 계단 함수

0을 넘으면 1을 출력하는 계단 함수를 구현

```python
import numpy as np
import matplotlib.pyplot as plt # 그래프를 구현해주는 라이브러리

def step_function(x):
    return np.array(x>0 , dtype=int) # 0보다 크면 True(1) 리턴

x = np.arange(-5,5,0.1) # -5 부터 5 까지 0.1 간격으로 넘파이 배열을 생성
y = step_function(x)
plt.plot(x,y) # 그래프 그리기 
plt.ylim(-0.1, 1.1) # y 축 범위 지정
plt.show() 
```

## 시그모이드 함수


```python
import numpy as np
import matplotlib.pyplot as plt
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

x = np.arange(-5,5,0.1)
y = sigmoid(x)
plt.plot(x,y)
plt.ylim(-0.1, 1.1)
plt.show()
```
