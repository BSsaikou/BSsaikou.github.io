## 함수

```python
def print_list(a): # 함수
  for i in a: 
    print(i)

a_list = [3, 4, 7, 9, 2, 1]

print_list(a_list)

# 3, 4, 7, 9, 2, 1

x = 1 # 전역변수
y = 3

def sum(a, b): # 매개변수

q = a + b # 지역변수
  return q

print(sum(x, y))

# 4
```

## 람다

x = 1

  return a + b
